# Security Policy

## Supported Versions

`@pane-craft/react` follows semver. Security fixes are applied to the latest minor release line.

## Reporting a Vulnerability

To report a vulnerability, use GitHub's private vulnerability reporting:

https://github.com/pane-craft/pane-craft/security/advisories/new

Please use private reporting and do not report security issues via public GitHub issues, discussions, or pull requests.
